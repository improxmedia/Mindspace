# 🚀 Host MindSpace FOREVER, for FREE — GitHub Pages

This folder is **ready to upload**. The link you get **never expires** (free permanently),
unlike the temporary Netlify Drop. No coding, no payment. ~5 minutes.

> Upload **everything in this folder**: `index.html`, `404.html`, `manifest.json`,
> `icon-192.png`, `icon-512.png`, and the hidden `.nojekyll` file.

---

## STEP 1 — Make a free GitHub account
1. Go to **https://github.com/signup**
2. Pick a username (this becomes part of your link), email, password. It's free.

## STEP 2 — Create a repository (a "folder" on GitHub)
1. Click the **+** (top right) → **New repository**.
2. **Repository name:** `mindspace`
3. Set it to **Public** (required for free Pages — but remember: only the app *code* is public, NEVER your data).
4. Click **Create repository**.

## STEP 3 — Upload these files
1. On the new repo page, click **"uploading an existing file"** (a link in the middle).
2. **Drag in all the files from this folder**:
   - `index.html`
   - `404.html`
   - `manifest.json`
   - `icon-192.png`
   - `icon-512.png`
   - `.nojekyll`  ← if you can't see/drag this hidden file, it's okay to skip it; the app still works.
3. Scroll down, click **Commit changes** (green button).

## STEP 4 — Turn on GitHub Pages (this makes it live)
1. In your repo, click **Settings** (top tab).
2. Left menu → **Pages**.
3. Under **"Branch"**, choose **main**, folder **/(root)** → click **Save**.
4. Wait about 1 minute. Refresh. You'll see:
   **"Your site is live at https://YOURUSERNAME.github.io/mindspace/"**

🎉 **That link is your app, forever.** Bookmark it.

---

## STEP 5 — Put it on your iPhone (Home Screen app)
1. Open the link in **Safari**.
2. Tap **Share (□↑)** → **Add to Home Screen** → name it "MindSpace" → **Add**.
3. It opens full-screen with the moon icon and your passcode lock.

## STEP 6 — One-tap 🛡 Shield (iPhone / Apple Watch)
1. Open the **Shortcuts** app → **+** → **Add Action** → search **Open URL**.
2. Paste your link with `#urge` on the end:
   `https://YOURUSERNAME.github.io/mindspace/#urge`
3. Name it **"Shield"** → add to **Home Screen / Apple Watch / Action button**.
4. One tap → it opens straight into the URGE flow (gratitude + protocol), before you fall.

---

## 🔐 Privacy (important)
- GitHub stores only the **app code** (same for everyone). It does **NOT** store your data.
- Your moods, captured thoughts, journal and gratitude live **only in your phone's browser** (local storage).
- Nothing about your struggle is ever uploaded. Use **Settings → Export backup** to keep a copy.

## ✏️ To update the app later
Just upload a new `index.html` to the same repo (it replaces the old one). The link stays the same.

## ❓ If the page looks blank for a minute
GitHub Pages can take 1–2 minutes the first time. Refresh. If still blank, check that
`index.html` is in the **root** of the repo (not inside another folder).
